# Friend-Finder

## What is Friend-Finder?
- Friend-finder is a mock dating site using node.js to create a full-stack web application.

## Setup
- Run npm install
    + This will install the following node.js applications:
        - Express.js
        - Body-parser
        - Path
        - Materialize
        - jQuery
